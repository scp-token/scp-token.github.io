## 1. プロジェクト概要

SCP Token（シンボル：SCP）は、技術およびコミュニティ実験として作成された標準的なブロックチェーントークンです。

プロジェクトの目的：

* 分散型トークン展開の探求
* オープンソースのスマートコントラクト開発の実践

* ブロックチェーン基盤のデプロイと検証の学習

本プロジェクトは投資助言、金融商品、証券を示唆するものではありません。内在的価値や保証されたユーティリティはありません。

SCP Token は 3 つの基本原則に基づいています：

### 1. 透明性
